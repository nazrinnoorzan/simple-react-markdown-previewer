# Simple React Markdown Previewer

Based on the Codepen project here https://codepen.io/freeCodeCamp/pen/GrZVVO

This project uses -

1. ReactJS (with React Hooks)
2. Create React App
3. Marked npm package

## How to run project?

1. Open console, run command:

##### `npm install`

2. Next, run command:

##### `npm start`
